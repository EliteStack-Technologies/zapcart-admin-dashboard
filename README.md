# ZapCart Admin Dashboard

Comprehensive business management dashboard for products, offers, and inventory.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn UI
- Tailwind CSS
- Supabase
- Framer Motion
- Recharts
- Socket.io Client

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Key Features

- **Product Management**: Create, update, and manage your inventory.
- **Offer Management**: Design and track promotional offers.
- **Analytics**: Visual data representation using Recharts.
- **Real-time Notifications**: Integrated with Socket.io.
- **Secure Backend**: Powered by Supabase.

## Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the production-ready application.
- `npm run lint`: Runs ESLint for code quality checks.
- `npm run preview`: Previews the build locally.
